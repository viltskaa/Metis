# **Подготовка к запуску**
1. Создание вирутальной среды  
```python -m venv venv```
2. Активация виртуальной среды  
```venv\Scripts\activate```
3. Загрузка зависимостей  
```pip install -r requirements.txt```

# **Запуск сервера**
1. Запуск сервера  
```flask run```
2. Инициализация базы данных  
```flask init-db```
